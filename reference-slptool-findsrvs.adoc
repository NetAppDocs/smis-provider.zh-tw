= slptool findsrvs
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
"shlptool findsrvs"命令可查找在網絡上運行的WBEM服務。

.語法
"lptool findsrvs _service_"

.位置
「C：\Program Files（x86）\NetApp\SMI\PEGUS\Bin」

.權限層級
具有有效使用者名稱和密碼的使用者

.參數
* *服務_*
+
指定服務類型。



.範例
