= SI清單
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
「mis list」命令會顯示新增的儲存系統。

SI清單

[-t {* http *|* https *｝]

「C：\Program Files（x86）\NetApp\SMI\PEGUS\Bin」

具有有效使用者名稱和密碼的使用者

.參數
* 「*」[-t｛http | https｝]*
+
要使用的傳輸協定：HTTPS（預設）或HTTP



「mis list」命令及其輸出：

[listing]
----
smis list
ONTAP_FilerData.hostName="10.16.180.122",port=80
----
