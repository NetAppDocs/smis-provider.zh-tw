= 使用NetApp SMI-S Provider
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
NetApp SMI-S Provider可讓您更輕鬆地管理及監控儲存系統、以及管理儲存系統的LUN和磁碟區。

您可以使用NetApp SMI-S Provider、使用System Center 2016 - Virtual Machine Manager或System Center 2016 - Virtual Machine Manager來管理儲存控制器。
