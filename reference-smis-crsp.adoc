= SMI系統
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
「mis crsp」命令說明NetApp SMI-S Provider支援的已登錄CIM子設定檔、包括NetApp SMI-S Provider子設定檔。

SMI系統

[-t {* http *|* https *｝]

「C：\Program Files（x86）\NetApp\SMI\PEGUS\Bin」

具有有效使用者名稱和密碼的使用者

.參數
* 「*」[-t｛http | https｝]*
+
要使用的傳輸協定：HTTPS（預設）或HTTP
