= slptool
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以使用「shlptool」命令來顯示有關WBEM服務的資訊。

.語法
「lpdool [選項]_子 命令_」

.位置
「C：\Program Files（x86）\NetApp\SMI\PEGUS\Bin」

.權限層級
系統管理員（Windows）

.選項
* *-I*
+
指定一個或多個介面。

* *-l*
+
指定語言標記。

* *至*
+
指定範圍清單（以逗號分隔）。

* *-u*
+
指定一個介面。

* *-v*
+
顯示"slptool"和OpenSLP"的版本。
