= 指定 ONTAP API 呼叫逾時值
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
SMI-S Provider 可將 ONTAP API 呼叫至儲存系統。根據預設， ONTAP API 呼叫逾時為 300 秒。您可以將逾時設定為60到300秒之間的值。

.開始之前
* 您必須已擁有系統管理員的登入認證。
* 您必須已以系統管理員身分登入主機系統。


.步驟
. 使用新的逾時值（以秒為單位）建立名為「ONTAPI_timeout_SEC」的系統或使用者環境變數。
+
如需建立環境變數的相關資訊、請參閱Windows文件。
